### Response Fields Description

#### Listing Object
- Contains detailed listing information including material specifications, dates, and status
- All fields from the original listing are preserved in their original format
- Additional fields specific to admin view are included

#### Seller Information Object
- `fullName`: Combined first and last name of the listing creator
- `company`: Name of the company associated with the listing

#### Material Information Object
- Condensed view of key material details:
  - `materialName`: Type of material being listed
  - `country`: Location of the material
  - `currency`: Currency used for the transaction
  - `packaging`: How the material is packaged
  - `capacityPerMonth`: Monthly capacity
  - `materialWeightWanted`: Total weight of material wanted

#### Bid Status Object
- Current status information:
  - `status`: Current listing status (e.g., PENDING, APPROVED, REJECTED)
  - `state`: Current listing state (e.g., PENDING, ACTIVE, REJECTED)
