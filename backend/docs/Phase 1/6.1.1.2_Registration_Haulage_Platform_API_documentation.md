# 6.1.1.2 Registration - Haulage Platform API Documentation

This API handles the initial user registration process for Haulier Platform users.

## API Endpoint

```
POST /register-haulier
```

## cURL Example

```bash
curl -L -X POST 'https://wastetrade-api-dev.b13devops.com/register-haulier' \
-H 'Content-Type: application/json' \
--data-raw '{
  "email": "newuser1@example.com",
  "password": "securepassword", 
  "firstName": "John",
  "lastName": "Doe",
  "prefix": "Mr.",
  "jobTitle": "Haulier Manager",
  "phoneNumberUser": "1234567890",
  "mobileNumberUser": "0987654321",
  "whereDidYouHearAboutUs": "Online",
  "companyName": "Waste Hauling Co",
  "registrationNumber":"21311232",
  "vatNumber": "GB123456789",
  "vatRegistrationCountry": "UK",
  "addressLine1": "123 Haulier St",
  "addressLine2": "Unit 4",
  "postalCode": "AB1 2CD",
  "city": "London",
  "stateProvince": "England",
  "country": "United Kingdom",
  "phoneNumberCompany": "9876543210",
  "mobileNumberCompany": "0765432109",
  "fleetType": ["Truck", "Van"],
  "areasCovered": ["London", "Manchester"],

    {
      "documentType":"carrier_license",
      "documentUrl":"https://example.com/document.pdf",

    }

```

## Conditional Fields

All expiryDate must be provided in DD/MM/YYYY format

Important: When including documentUrl and expiryDate, the arrays must have the same length. Each license URL must have a corresponding expiry date at the same index
