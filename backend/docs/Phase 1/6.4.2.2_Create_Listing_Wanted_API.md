## Note about fields
base on this link about material options `https://docs.google.com/spreadsheets/d/1_RqEAiolZMBvuzi-A5oO3d2dpISdIX5mgrBNy0z5bpY/edit?gid=1611260389#gid=1611260389`
If field option is N/A. please don't insert field into requestBody
for example 
- `materialType` is `efw`, don't provide `materialItem`, `materialForm` & `materialGrading` 
- `materialType` is `metal`, don't provide `materialForm` & `materialGrading`

## Note 
In Design, there is input for materialWeight & metric (LBs, Kg, MT). Please convert metric LBs & Kg into MT (1 MT = 1000 Kgs = 2204.62263 Lbs) before send in requestBody `materialWeightWanted`
