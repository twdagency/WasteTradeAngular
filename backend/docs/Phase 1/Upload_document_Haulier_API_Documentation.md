# Upload document Haulier API Documentation

This API handles upload files on Upload documents when create account Haulier.

## API Endpoint

```
POST /upload-file-haulier
```

## cURL Example

```bash
curl -L -X POST 'http://127.0.0.1:3000/upload-file-haulier' \
-H 'accept: application/json' \
-F 'file=@"/C:/Users/OMEN/Downloads/download.png"' \
```
